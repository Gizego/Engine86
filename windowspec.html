Gizego Engine
Window class specification

Funcion list in GizegoEngine::Window:

Window(GizegoEngine::Size size, std::wstring title)   Creates a window with this parameters. To show for the first time it use Create().

GizegoEngine::Renderer* Create(GizegoEngine::GLVersion version = GizegoEngine::GLVersion(3, 0))   Shows the window for the first time. Allways call after the constructor and never't delete the return value.

bool IsOpen()   Returns if the window is open or not.

void Update()   Call this every time in your game loop.

void SwapBuffers()   Call this every time in your rendering loop.

GizegoEngine::Size GetSize()   Returns the client size of the window.

std::wstring GetTitle()   Returns the title of the window.

void SetTitle(std::wstring title)   Sets the title of the window.

void Close()   Closes the window.

void Hide()   Hides the window.

void Show()   NOT A REPLACEMENT FOR CREATE. This will only show it after you hide it.

void Maximize()   Maximizes the window.

void Minimize()   Minimizes the window.

bool IsMaximized()   Checks if the window is maximized.

bool IsMinimized()   Checks if the window is minimized().

void SetVSyncState(bool state)   Turns VSync on or off.

bool GetVSyncState()   Gets the VSync state of the window.

void FullScreen(GizegoEngine::Display* display)   Sets the window to windowed full screen mode and resizes it to fit the monitor's resolution.

void FullScreen(GizegoEngine::Display* display, GizegoEngine::Size size)   Sets the window to windowed full screen mode but you can set its size.

void SetIcon(GizegoEngine::Image* icon)   Sets the window's icon.

void Resize(GizegoEngine::Size size)   Sets the client size of the window.

GLFWwindow* GetHandle()   DO NOT USE. This is a GLFW thing and it's only public because other classes need it.

static GizegoEngine::Window* GetMainWindow()   The window class is supposed to be used as a singleton. This will return the only instance of it.

GizegoEngine::vec2 GetWindowPosition()   Gets the window position in the screen.

void SetPosition(GizegoEngine::vec2 pos)   Sets the cindow position in the screen.
