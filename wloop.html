Window and game loop tutorial

Creating the window:

The first thing you should do before you start your application is #include "include/GizegoEngine.h"
Create a function to handle exceptions.
void ExceptionHandler(std::wstring exception, GizegoEngine::ExceptionGravity gravity) {
}

As the main focus of this game engine is liberty, you can handle exceptions the way you want. Examples are logging them to the console, throwing std::exceptions or showing a message box. The variable "gravity" can be GizegoEngine::ExceptionGravity::Error or GizegoEngine::ExceptionGravity::Warning. Errors might stop your program from advancing and you should exit it if you get one.

The first thing to do in main is calling GizegoEngine::InitializeEngine(ExceptionHandler /*the void you created*/)

Then, you can create the window.
unsigned int width = 500, height = 500;

This will only allocate memory for the window and set the creation parameters.
GizegoEngine::Window* window = new GizegoEngine::Window(GizegoEngine::Size(width, height), L"Window title");
Don't forget to delete the window at the end of your program.

THE WINDOW CLASS IS A SINGLETON. DO NOT CREATE MORE THAN ONE. This is the function that will actually create a context and show it.
GizegoEngine::Renderer* renderer = window->Create();
I recommend you don't change the OpenGL version parameter. 3.0 is the minimum required. Also, do not delete the renderer in the end of your program.

Now, we can move on to the game loop.
while (window->IsOpen()) {
window->Update();
window->SwapBuffers();
}

Then, after the game loop, the window can be deleted and the engine terminated.
delete window;
GizegoEngine::CloseEngine();

If you want your game to be on fullscreen use this before your game loop:
GizegoEngine::Display* display = new GizegoEngine::Display();
window->FullScreen(display);
delete display;
